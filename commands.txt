================================================================================
  1. INSTALLATION & SETUP
================================================================================

  Prerequisites:
    - Python 3.10+
    - Ollama installed and running (https://ollama.com)
    - At least one model pulled (e.g. ollama pull deepseek-v3.1:671b-cloud)

  Install (from project root):
    cd C:\Users\Sinwa\Pictures\ClaudeAI\claw-agent
    pip install -e .

  Verify:
    claw --help
    claw --version


================================================================================
  2. START / STOP / RESTART
================================================================================

  --- Start Ollama ---
    ollama serve                              # Start Ollama server
    ollama serve &                            # Start in background (Linux/Mac)
    Start-Process ollama -ArgumentList serve   # Background (Windows PowerShell)

  --- Start CLI Agent ---
    claw                                      # Interactive mode (default model)
    claw --model qwen2.5:7b                   # Use specific model
    claw prompt "summarize this repo"         # One-shot mode (non-interactive)

  --- Stop CLI Agent ---
    /quit                                     # Inside the CLI
    Ctrl+C                                    # Force quit

  --- Restart CLI Agent ---
    /quit                                     # Exit first
    claw                                      # Start again (fresh session)
    claw --model deepseek-v3.1:671b-cloud     # Restart with different model

  --- Start VS Code Extension ---
    1. Open VS Code
    2. Ctrl+Shift+P → "Claw Agent: Open Chat"
    3. Or click the Claw icon in the sidebar
    Bridge auto-starts when you open the chat panel.

  --- Stop VS Code Extension ---
    Close the Claw Agent chat panel.

  --- Restart VS Code Extension ---
    Ctrl+Shift+P → "Developer: Reload Window"

  --- Start Chrome Extension ---
    1. Go to chrome://extensions
    2. Enable "Developer mode" (top right)
    3. Click "Load unpacked" → select chrome-extension/ folder
    4. Click the Claw icon in toolbar → opens side panel

  --- Stop Chrome Extension ---
    Close the side panel, or disable in chrome://extensions.

  --- Restart Chrome Extension ---
    chrome://extensions → click reload (↻) on Claw Agent

  --- Stop Ollama ---
    Ctrl+C in the ollama serve terminal
    taskkill /IM ollama.exe /F               # Force kill (Windows)
    killall ollama                            # Force kill (Mac/Linux)


================================================================================
  3. CLI SLASH COMMANDS (22 commands)
================================================================================

  --- Core ---
  /help                   Show all available commands
  /model <name>           Switch to a different Ollama model
  /models                 List all available Ollama models
  /tools                  List all 22 agent tools with descriptions
  /cost                   Show token usage, timing, turns stats
  /compact                Compress conversation to save context window
  /clear                  Clear conversation and reset state

  --- Sessions ---
  /save                   Save current session to disk
  /sessions               List all saved sessions
  /resume <id>            Resume a previously saved session
  /delete <id>            Delete a saved session
  /export                 Export conversation to markdown file

  --- Git & Workspace ---
  /diff                   Show git diff (what the agent changed)
  /status                 Show workspace info: path, git branch, project type
  /undo                   Undo last file write (git checkout last changed file)

  --- Configuration ---
  /config                 Show current agent configuration table
  /version                Show Claw Agent version, Ollama version, Python version
  /doctor                 Run 8 diagnostic checks (Ollama, tools, connectivity)
  /permissions            Show tool permission summary (gated/allowed)
  /memory                 Show context window usage bar
  /init                   Generate .claw config file for the project

  --- Debug ---
  /bug                    Dump debug info for bug reports

  --- Control ---
  /quit                   Exit the CLI

================================================================================
  10. TROUBLESHOOTING
================================================================================

  Agent not responding?
    → Check Ollama is running: ollama list
    → Run /doctor in the CLI for full diagnostics

  Chrome extension 403 error?
    → Set OLLAMA_ORIGINS=* as environment variable
    → Windows: System Properties → Environment Variables → User → New
    → Or PowerShell: [System.Environment]::SetEnvironmentVariable("OLLAMA_ORIGINS", "*", "User")
    → Restart Ollama after setting

  VS Code extension not connecting?
    → Ctrl+Shift+P → "Developer: Reload Window"
    → Check Output panel → select "Claw Agent"

  Too many tool calls / looping?
    → Circuit breakers auto-stop after errors
    → /stop (Chrome) or Ctrl+C (CLI) to force stop
    → /clear to start fresh

  Out of context window?
    → /compact to compress conversation

  MEMORY.md or SOUL.md not loading?
    → Must be in the project root (or up to 3 parent dirs)
    → Run /config in CLI to confirm project context loaded
    → File must be under 4K characters
    → /clear to reset completely
