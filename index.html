CLI Agent — Terminal Coding Assistant

Install the full coding agent for your terminal. Works like Claude Code.

Requirements

  • Python 3.10 or newer
  • Git
  • pip (comes with Python)

Quick Install (any OS)

pip install git+https://github.com/kdhdjdk090/claw-agent.git

Development Install (from source)

git clone https://github.com/kdhdjdk090/claw-agent.git
cd claw-agent
pip install -e .

Windows Quick Start

# Option 1: Double-click start-claw.cmd in the project folder

# Option 2: Command line
pip install git+https://github.com/kdhdjdk090/claw-agent.git
claw

Linux / macOS Quick Start

pip install git+https://github.com/kdhdjdk090/claw-agent.git
claw

Usage

# Interactive REPL mode
claw

# One-shot mode
claw "write a python web scraper"

# Specify model
claw --model deepseek-r1:671b

# Print-only mode (no file edits)
claw --print "explain this code"

# Auto-approve all tool calls
claw --yolo "refactor this project"

CLI Commands

/help         — Show all commands
/model <name> — Switch model
/tools        — List available tools
/skills       — View skills
/cost         — Session cost tracker
/save         — Save session
/resume <id> — Resume session
/diff         — Show recent changes
/undo         — Undo last action
/config       — View configuration
/quit         — Exit

Cloud Mode (no Ollama needed)

# Set your API key
# Windows PowerShell:
$env:DEEPSEEK_API_KEY="your-key-here"

# Linux/macOS:
export DEEPSEEK_API_KEY="your-key-here"

# Then run:
claw

Local Mode (with Ollama)

# Terminal 1: Start Ollama
ollama serve

# Terminal 2: Pull a model and run
ollama pull deepseek-r1:32b
claw
Pre-configured API keys are included for OpenRouter and Alibaba Cloud. For local mode, install Ollama from ollama.com.

Chrome Extension — Browser AI Agent

Add Claw AI as a side panel in Chrome with 40+ browser control tools.

Install Steps

  1. Clone or download the repository:
    git clone https://github.com/kdhdjdk090/claw-agent.git
  2. Open Chrome and go to chrome://extensions/
  3. Enable Developer mode (toggle in top-right corner)
  4. Click Load unpacked
  5. Select the chrome-extension/ folder from the cloned repo
  6. Click the puzzle icon in Chrome toolbar and pin Claw Agent

Features

  • Side panel AI chat in any tab
  • 40 browser control tools (click, fill, navigate, extract)
  • Context menu: right-click text → Explain / Fix Code / Rewrite / Summarize
  • Quick actions: Summarize page, Extract links, Find code, Explain
  • Streaming responses
  • Session persistence
  • Model selection dropdown
  • Works with OpenRouter Cloud API (no local setup)

Extension Files

chrome-extension/
├── manifest.json     # Extension config (Manifest V3)
├── background.js     # Lifecycle & context menus
├── content.js        # Page extraction & browser tools
├── sidepanel.html    # Chat UI
├── sidepanel.js      # Chat logic
├── sidepanel.css     # Styles
└── icons/            # 16, 32, 48, 128px icons
API keys are pre-configured in the extension. No manual key setup needed.

VS Code Extension — In-Editor AI Agent

Get Claw AI inside VS Code as a chat panel with code actions.

Method 1: VSIX Package (Recommended)

# Step 1: Package the extension
cd vscode-extension
npm install -g @vscode/vsce
vsce package
# Creates: claw-agent-2.0.0.vsix

# Step 2: Install in VS Code
# Extensions (Ctrl+Shift+X) → ⋯ menu → Install from VSIX
# Select claw-agent-2.0.0.vsix

Method 2: Development Mode

# Open the extension folder in VS Code
cd vscode-extension
code .

# Press F5 to launch Extension Development Host
# A new VS Code window opens with Claw Agent loaded

Keyboard Shortcuts

Ctrl+Shift+A    — Open Agent Chat (Windows/Linux)
Cmd+Shift+A     — Open Agent Chat (macOS)

Commands

Claw: Open Agent Chat     — Open the chat panel
Claw: Quick Prompt        — Inline prompt
Claw: Explain Selection   — Right-click → Explain code
Claw: Fix Selection       — Right-click → Fix code

Settings

{
  "claw.apiMode": "cloud",
  "claw.model": "openai/gpt-4o-mini",
  "claw.openrouterApiKey": "",
  "claw.ollamaUrl": "http://localhost:11434"
}
